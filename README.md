This repository demonstrates a collaborative workflow between a human researcher and an AI assistant (Codex/Copilot) to create a professional profile. It specifically highlights my research as a Graduate Research Assistant at the JHU Center for Digital Health and AI, focusing on the integration of consumer-grade data with medical sensors to predict myocardial infarction and atrial fibrillation.

Steps taken throughout the process:

- Initialized the repository and created the initial file structure via the Mac Terminal.
- Staged changes using git add after the AI generated the initial draft and after I completed my manual refinements.
- Committed my progress using git commit -m with descriptive messages to track the evolution from an AI-generated draft to a human-verified final version.
- Push the local project to GitHub to ensure version parity and accessibility.
